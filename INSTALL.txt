Instrucciones para realizar la instalación del Programa.

1. Se debe contar con Visual Studio Code con extensión de c++.
2. Debe clonar el repositorio en su computador.
3. Debe abrir el archivo Proyecto2.c++ en el Visual Studio Code.
4. Debe tener en la misma carpeta el archivo que tiene la información de los guardianes (guardians.conf) y las ciudades (cities.conf).
5. Debe abrir una nueva terminal y escribir "./Proyecto2".
